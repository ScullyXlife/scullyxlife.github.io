Iron Translator

A Discord translator bot hosted by ScullyXlife. It translates live in Discord, runs on local translation models, and does not store user messages or translated text.

Install Iron Translator View Commands

Guild Setup

  1. Use Install Iron Translator and choose the Discord server you manage.
  2. In that server, run /subscription action:install.
  3. Use the Subscribe button from that command. That checkout is tied to your guild ID.
  4. After payment clears, the bot unlocks that guild automatically. This can take about a minute.

Requirements

  • You need Administrator or Manage Server permission to unlock a guild.
  • The bot needs View Channels, Send Messages, Read Message History, and app command access.
  • Use the bot-generated Subscribe button. The raw payment link does not attach your guild ID.
  • Translation output is private by default.

Commands

  • /translate text:... translates text. Source language auto-detects unless you choose one.
  • /languages shows the current languages installed on the bot.
  • /translator_status shows runtime status and whether your guild is unlocked.
  • /subscription action:install creates the guild-tied install and subscription flow.
  • /subscription action:status checks subscription/guild binding status.
  • /subscription action:release releases a subscription-bound guild when supported.
  • /auto_translate action:set lets a server manager turn on public auto-translation for a channel.
  • /auto_translate action:set role:@German source:German target:English bidirectional:true follows a language role across the guild.
  • /auto_translate action:list and /auto_translate action:remove manage configured auto-translate channels and roles.
  • Apps -> Translate message opens the right-click message translator.
